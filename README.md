# ML Engineer Portfolio

A modern, responsive portfolio website showcasing ML/AI projects and expertise.

## Features

- 🎨 Modern, clean design
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Smooth animations and transitions
- 🚀 Fast loading and optimized
- 💼 Project showcase with modals
- 📧 Contact form
- 🎯 SEO friendly

## Projects Showcased

1. **Feedback-to-Fine-Tuning Pipeline** - End-to-end ML pipeline
2. **Healthcare AI Chatbot** - Production healthcare AI assistant
3. **Automated Model Evaluation System** - Evaluation framework
4. **LLM Fine-Tuning Platform** - Fine-tuning platform

## Technologies

- HTML5
- CSS3 (with CSS Grid & Flexbox)
- Vanilla JavaScript
- No frameworks or dependencies

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Import to Vercel
3. Deploy automatically

### Netlify

1. Drag and drop the `portfolio` folder
2. Or connect GitHub repository

### GitHub Pages

1. Push to GitHub
2. Enable GitHub Pages in repository settings
3. Select main branch

## Customization

1. **Update Projects:** Edit project cards in `index.html`
2. **Change Colors:** Modify CSS variables in `styles.css`
3. **Add Projects:** Add new project data in `script.js`
4. **Update Links:** Change demo links in project data

## File Structure

```
portfolio/

├── styles.css      # All styles
├── script.js       # JavaScript functionality

```
